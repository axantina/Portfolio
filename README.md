## Embedded IoT

Embedded systems and IoT projects using Arduino, ESP32, LoRa, sensors, and hardware control.

### [Embedded IoT Smart Greenhouse](Embedded%20IoT/embedded-iot-smart-greenhouse)

An Arduino-based smart greenhouse system.

Main features:

* Soil moisture monitoring
* LDR light detection
* LCD I2C display
* Keypad input
* Relay and water pump control
* LED status indicator

### [LoRa WSN Monitoring System](Embedded%20IoT/lora-wsn-monitoring-system)

A LoRa-based wireless sensor network monitoring system using ESP32.

Main features:

* ESP32 and LoRa communication
* DHT11 temperature and humidity sensor
* Soil moisture sensor
* SD card data logging
* PHP and MySQL backend
* QoS analysis in line-of-sight conditions
